# Καθημερινά V13.5.5 — Instant Quiz Feedback + Calm Milestones

Single-file offline PWA for daily Greek conversation practice, with an A1/A2 Ellinomatheia exam-prep scaffold.

## Current build

V13.5.5 improves the quiz experience without changing the SRS math. Quiz answers now give immediate per-question feedback, and milestone quizzes show a calm adult marker when passed.

## Quiz feedback

- Choose an answer and see immediate Correct / Review feedback.
- Correct answer is shown instantly after a miss.
- Final scoring still feeds the existing SRS once at quiz completion.
- No new Greek is generated by the quiz layer.

## Calm milestone marker

Passing a milestone quiz at 75% or higher stores a quiet readiness marker such as **A1 ready** or **A2 ready**. This is a practice milestone, not an official exam score.
